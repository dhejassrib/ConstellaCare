# ConstellaCare

## Overview

ConstellaCare is an AI-powered wellbeing platform that supports both individuals navigating health journeys and the people who care for them.

By combining emotional wellbeing tools, AI guidance, care planning, and collaborative support features, ConstellaCare helps users feel more connected, supported, and empowered throughout their healthcare experience.

---

## Problem

Health journeys can be emotionally overwhelming for both care recipients and caregivers.

While healthcare systems often focus on physical treatment, emotional wellbeing, communication, and support coordination are frequently overlooked.

---

## Solution

ConstellaCare provides a shared digital space that helps users:

* Track wellbeing and emotions
* Reflect through guided journaling
* Organize appointments and care tasks
* Strengthen communication
* Access AI-powered support and insights

---

## Key Features

⭐ Shared Constellation Progress System

😊 Biometric Mood Scanner with personalized wellbeing suggestions

📅 Health Journey Timeline

✅ Personalized Checklists

📝 Private Reflection Journal

🤝 Care Circle Support Network

💬 AI Communication Assistant

🌙 Light & Dark Mode Experience

## Getting Started

To run it on localhost:
Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
